WHAT IS JAVA?

Java is a widely used programming language expressly designed for use in the distributed environment of the internet. It is the most popular programming language for Android smartphone applications and is also among the most favored for the development of edge devices and the internet of things.

Let's start here.

DID YOU GET ALL THAT?

It's okay if you didn't. The idea here is to get you familiar with some of the core concepts of Java and programming in general. Not getting some of the finer details at this point is perfectly fine. If you're new to programming don't worry. Java is a great first language to learn since it's core concepts will easily translate to other languages.

Enough chat, let's install Java so we can write our first program!

  • Install the Java JDK

    First step is to install the Java Development Kit. This will allow you to compile and run your java programs.


    Java JDK

  • Add to system variables

    This will allow you to compile and run your Java files through the command prompt. Go to start menu and type for Environment Variables. There should be an option that let's you edit them.

  • Test

    Check the version types for java using the command prompt. If you get a version number for both "java -version" and "javac -version" then you are good to go!

Hello World.

Alright, now that we have installed java and saw the version numbers in CMD, let's actually get to coding! It's tradition to write a "hello, world" program when learning any new language so let's do that.


Keep in mind that some of the syntax you write won't make any sense right now, don't worry. As you play around and learn more, you'll begin to pick up on what everything does, and why it does it. For now let's start by outputting to the screen.

01

The first thing to to notice about this code is line 1. This is a single line comment. Anything after the // on the same line will be seen as a comment.

02

On line 3 is where the class starts. Don't worry to much about this right now, just know that the class must be named the same as the file. Also note at the end of this line is a opening curly brace. The closing curly is on line 9. Anything inside these curly braces are considered part of the class.

03

One line 5 is where the main method starts. For now, know that the main method is where the program begins execution. Note the curly brace at the end of the line 5, this brace opens the main method and ends on line 8. Anything between these braces will be executed.

04

Line 7 is the only thing in our main method. In this case it will print out "Hello, world!" to the console and then the program will end.
